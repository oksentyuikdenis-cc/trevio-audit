## ✅ Що працює прямо зараз

- **Сайт живий**, зібраний з коміту, який задеплоєний (перевірено по SHA, не по «HTTP 200»).
- **Нуль мережевих запитів** — картку вшито в бандл. Сторінка відкривається без інтернету.
- **Пошук працює без сервера.** Три запечені аудити лежать у бандлі: набирайте *Trenitalia*, *Satispay*, *Notion* — відповідь миттєва, з телефона, без Wi-Fi і без квоти. Це головне для аудиторії з QR.
- **Телефон**: нуль горизонтального скролу, нуль елементів за краєм екрана (перевірено на 375×812).
- **Проєктор**: усі контрасти проходять WCAG AA, найгірший 5.12 при нормі 4.5.
- **Друк** (кнопка «Save this audit as PDF»): ~3 сторінки A4, цитата не рветься посеред речення, маркетингові заголовки лендингу на папір не потрапляють.

## 🔴 Що потрібно від вас

### 1. Ключ Mistral перестав діяти — HTTP 401

Був робочий, зараз сервіс відповідає `Unauthorized`. Ймовірно відкликаний разом зі старим.

Через це **картки перепечено на Gemini**, і на них стоїть штамп
`Gemini · US (and trains on free-tier prompts)` — що суперечить тезі «дані не покидають Європу».

**Дія:** створити новий ключ на [console.mistral.ai](https://console.mistral.ai) → вставити у `trevio-audit-server/.env` у рядок `MISTRAL_API_KEY=` → сказати мені, і я перепечу за хвилину. Цифри від цього не зміняться, зміниться рядок атрибуції — але саме він підпирає європейську тезу.

> Якщо не встигнете — це не блокер. Картки правильні, атрибуція чесна, просто не та, яку хотілося б показати під слайд про Європу.

### 2. Ключ SerpApi — для H-FARM через Google Maps

`SERPAPI_KEY=` у `.env` порожній. Без нього H-FARM недосяжний: у застосунку 21 оцінка й **нуль** написаних відгуків, усе живе на Maps.

Весь код Maps **доведено наскрізь** проти локальної підробки SerpApi — пагінація, недатовані рядки, підпис джерела, перевірка заземлення. Щойно ключ з'явиться, воно працює; першого запуску в залі не буде.

**Дія:** [serpapi.com/users/sign_up](https://serpapi.com/users/sign_up) → ключ у `SERPAPI_KEY=` (рядок 24, не в коментар).

**Запасний шлях без реєстрації:** скопіювати відгуки H-FARM з Maps у текстовий файл, один відгук на абзац, порожній рядок між ними, потім:

```bash
npm --prefix trevio-audit-server run fixture -- "H-FARM" --paste файл.txt
```

Шлях через вставку перевірено на італійському тексті: 7 випадків розбору, усі правильні.

### 3. Пошта у CTA — заглушка

`src/sections/Cta.tsx`, кнопка «Request access» → `hello@trevio.app`. Адреси не існує, листи відскакують. Свідомо не міняв: підставляти особисту gmail на публічний лендинг — ваше рішення.

## ▶️ Запуск демо

**Термінал 1 — сервер аудиту:**

```bash
npm --prefix trevio-audit-server start
```

Успіх — надрукує `trevio-audit-server on http://localhost:8787`, назве провайдера і перелічить запечені цілі. Далі просто стоїть.

**Термінал 2 — сайт для залу:**

```bash
npm --prefix trevio-murano run stage
```

Успіх — надрукує два URL. **Читайте вголос `Network:`** — саме він відкривається з телефонів у залі. `Local:` працює тільки на вашому ноутбуці.

> Використовуйте `stage`, а не `dev`. `dev` слухає лише localhost, і телефони до нього не достукаються. `stage` вмикає пошук і прив'язується до мережі.

**Не запускайте** `npm run fixture` під час демо — воно витрачає квоту, від якої залежить живий пошук.

## 🆘 Якщо щось падає

**Провайдер уперся в ліміт посеред демо.**
Нічого не робіть. Картка Trenitalia вже на екрані — вона у бандлі, не тягнеться. Тисніть чіпи *Trenitalia / Satispay / Notion*: вони відповідають з диска, **без жодного виклику моделі**. Живий пошук — це підсилення, а не залежність.

**Хтось шукає компанію без застосунку.**
Система чесно каже, що не знайшла, і не вигадує. Це можна повернути на свою користь: більшість організацій світу не мають застосунку — саме тому в проєкті є Google Maps і вставка тексту.

**Питання «звідки знаєте, що AI не вигадав цифри?»**
Відповідь на екрані, під карткою, дослівно:

> *Every figure above is computed from the review rows; the model decides only which reviews belong together and writes the summary, so there is no path by which a number it invented reaches this card.*

Підсилення: модель фізично не має поля для числа у своїй схемі відповіді. Після збірки окрема перевірка шукає цитати дослівно в корпусі й **валить прогін**, якщо не знаходить, — не рендерить із застереженням, а валить.

## ⚠️ Що сказати самим, не чекаючи питання

- Джерел кілька, не двадцять п'ять. X, LinkedIn і Reddit недоступні **з причин платформ** — назвати поіменно сильніше за мовчання.
- Hacker News підключений, **виміряний і вимкнений**: із 261 коментаря зі словом «Notion» лише 5 виявились справжнім досвідом користування. Це сильніша теза, ніж і показати мотлох, і промовчати.
- Живий прогін недетермінований — два запуски дають трохи різні теми. Тому на сцені запечені цілі.
- Немає моніторингу, історії, акаунтів. Це разовий аудит.
- Немає ARR і churn: для цього потрібна CRM, а не публічні відгуки. На картці навмисно **немає жодної суми грошей**.
